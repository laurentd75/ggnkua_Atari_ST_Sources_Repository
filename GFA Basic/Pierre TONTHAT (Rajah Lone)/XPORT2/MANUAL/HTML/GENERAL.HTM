Mode d'emploi


Préambule

    Xport2 est un logiciel de transfert de données (textes) développé pour Atari ST et Atari Portfolio.

    Le transfert se fait par le port série, il faut donc posséder le module "série" du Portfolio.

    D'autre part, un accélérateur port série du ST (type HSMODEM) permet d'agrandir les buffers série d'import-export et de sécuriser les transferts de données. HSMODEM est donc conseillé, mais pas indispensable.

    Quelque soit le tpe de transfert que vous effectuez, c'est la machine qui importe qui doit être activée en premier et être en attente de celle qui va envoyer les données.

    Avant tout transferts, veillez bien à configurer les paramètres du port série du Portfolio et à initialiser ce port.


Import : Portfolio vers ST

    Réglez la vitesse de transfert en bauds de part et d'autre.

  • Sur le ST : lancez l'importation. Vous avez 2 modes : vous pouvez dans le premier visualiser ce qui arrive et qui consomme beaucoup de temps machine, d'où perte d'octets si votre Atari fait autre chose en même temps (calcul POV ou module sonore). Dans le seconde mode, vous importez à l'aveugle, sans affichage, d'où une plus grande sécurité.

    Quand l'importation est lancée, Xport2 scrute et récupère toutes les données qu'il peut lire sur le port série du ST. L'arrêt se fait toujours manuellement avec un clic souris.

  • Sur le Portfolio : vous avez 2 modes d'envoi de vos fichiers. Le premier se fait sous le DIP OS avec 
    copy fichier.txt aux
    Le second se fait sous xterm1 que vous avez installé sur le Portfolio, lancez ce programme, faites (F3 ou Shift+F3) et choisissez le nom du fichier à envoyer, puis appuyez sur (Enter). Le transfert se fait alors.

  • Sur le ST, cliquez pour terminer la récupération des données et sauvez-le avec le sélecteur de fichier qui apparaît.


Export : ST vers Portfolio

  • Sur le ST : charger le fichier à envoyer.

  • Sur le Portfolio : malheureusement, je n'ai réussi qu'à faire marcher correctement l'export qu'avec xterm1, et seulement pour des textes. Donc lancez-le, faites (Shift+F2), choisissez le nom de fichier où xterm1 vers sauver les données, et faites (Enter).

  • Sur le ST : démarrez l'export. Si tout va bien et si c'est du texte, xterm1 affiche ce qu'il reçoit. L'export sur le ST se fait en tâche de fond. L'arrêt est automatique sur le ST (fin de fichier lu) ou manuel en cours de route (cliquez sur l'option prévue).

  • Sur le Portfolio : quand vous voyez que tout a été récupéré, quittez avec (F1).
