Installation

Blaise nécessite 1 Mo de mémoire vive pour fonctionner. Pour la cohabitation avec POV en mémoire, vous aurez besoin de 2 Mo, sinon il faudra quitter Blaise pour lancer POV.

Un disque dur est conseillé. Si vous n'en avez pas, un second lecteur de disquette serait utile.

  1. Installez Blaise n'importe où sur votre disque.

    Les nouveaux BLAISE.PRG, BLAISE.RSC doivent être côte à côte.

  2. Déterminez les chemins et l'éxécutable de POV (F2).

  3. Allez dans la fenêtre du viewer (icône oeil) et configurer les préférences pour les modules M&E du viewer interne. La gestion mémoire de Blaise à propos de ces modules est modifiable dans les préférences diverses (menu Options ou F3). Vous avez plusieurs choix possibles. Ceux conseillés sont :
    • Taille fixe avec le maxiumum de mémoire avec le PARX.MEM 5.5X (celui de Digital Lab) sous MagiC ou MiNT.
    • Taille élastique avec le PARX.MEM 5.0X (celui du pack de PARX).
    Si vous n'avez que le PARX.MEM 5.0X et que vous êtes sous MagiC : sélectionnez "initialisation à chaque appel".
    Si vous n'avez pas assez de mémoire, désactivez le viewer interne.

    La gestion mémoire du viewer interne sera valide au prochain lancement de Blaise. (N'oubliez pas de sauver la configuration).
