Les raccourcis-clavier

Pour les formulaires ou les fenêtres ayant du texte à taper, comme ça vous n'avez pas besoin de quitter votre aimable clavier pour votre souris baladeuse.

N'importe où:

    Shift+F1 : Sauver les options
    F2 : Définir les chemins pour POV
    F3 : Paramètres divers
    F4 : Définir les macros

    F5 : Editer un Script POV
    F6 : Editer un Include
    F7 : Editer une liste de commandes POV

    F8 : Shell interne
    F9 : Appel d'un viewer avec un fichier en paramètre
    F10 : Appel de POV, avec oui ou non appel d'un viewer (en choisir un qui reconnaisse le format de l'image créée par POV)

    Control+Q : Quitter Blaise
    Help : appeler la toolbox

    Control+U : Fermer la fenêtre
    Control+W : Agrandir ou rétrécir la fenêtre

Pour les fenêtres d'édition:
    Control+L : Charger
    Control+M : Merger
    Control+S : Sauver
    Control+P : Imprimer (GFA Llist donc attention au bug du Falcon cf STMag)
    Control+Backspace : Effacer texte ou bloc
    Control+A : Aller à la ligne
    Control+C : Chercher
    Control+V : Chercher encore
    Control+R : Remplacer
    Control+H : Sélectionner d‚but du bloc (le Haut)
    Control+B : Sélectionner fin du bloc (le Bas)
    Control+I : Insérer SCRAP au niveau du curseur.
    Control+J : Copier bloc dans clipboard
    Control+K : Déplacer bloc dans clipboard

    Insert : Insérer une ligne façon éditeur GFA
    Control+Del : Effacer une ligne également façon GFA

    Alt+valeur ASCII : marche sans problèmes, sans rien avoir tapé une ligne de code. (C'est bien, le GEM!) A vous les accents circonflexes et l'écriture en Hébreu. (sauf sous MagiC, bien sûr, mais c'est corrigé avec CKBD).

Pour le shell, les raccourcis du win-menu:
    Control+A : Ajouter une application
    Control+Del : Enlever l'application
    Control+I : Informations
    Control+L : Lancer
Pour le shell, la fenêtre Ajouter:
    Control+flèche : Changer d'icône
    Control+A : Ajouter l'application
    Control+F : Choisir l'application par le fileselector
Pour le shell, la fenêtre Information:
    Control+flèche : Changer d'icône
    Control+L : Appliquer et lancer
    Control+A : Appliquer
    Control+N : Annuler
    Control+F : Choisir le fichier
Pour la fenêtre Macros:
    Control+flèche : Changer de touche
    Control+P : imprimer la liste entière de macros
    Control+L : Charger un ensemble de macros
    Control+S : Sauver les macros de texte
Pour la fenêtre Lancer le viewer:
    Control+F : Choisir l'image
    Control+O : Ouvrir le viewer interne
    Control+L : Lancer le viewer externe
Pour la fenêtre Lancer POV:
    Control+F : Choisir le define
    Control+L : Do the rendering!
